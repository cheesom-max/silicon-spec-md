# Feature Known Issues Template

## Purpose
- Track unresolved feature-specific issues, limitations, and temporary workarounds.

## Usage Order
- Read during implementation, QA, and support triage when known constraints may affect outcomes.

## Intended Audience
- Engineers, QA, reviewers, and operators handling edge-case behavior.

## Out of Scope
- Canonical feature behavior ownership.
- Root-cause decision records.
- Full incident-response procedures.

## Canonical Scope
- Current unresolved issues tied to this feature package.
- Workaround guidance and owner/next-review metadata per issue.
